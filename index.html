About Me

I graduated from California State University, Fullerton in May 2017 with a Bachelor of Fine Arts degree in Graphic Design. I also received a certificate of completion from University of California, Irvine in July 2018 after finishing the Full Stack Web Development bootcamp. Most of my graphic design work is made through Adobe Illustrator and Adobe Photoshop. For print based designs, I have created beer labels, business cards, icons, logos and fliers. For web based designs, I have created online fliers, advertisements for web show interviews and thumbnail images for YouTube videos. Due to my experience with graphic design, I prefer to work in the front end when coding. This allows me to take advantage of my design skills when creating webpages and applications.

My hobbies include reading, watching movies, playing video games and above all, scuba diving. There is nothing I would rather be doing than scuba diving. I've dove inside a sunken ship, through dormant lava tubes, drifted along underwater currents, in the dark during the night and gone as deep as 100 feet below the surface. Most notably, I've dove with sea turtles, manta rays and hammerhead sharks!

Web Development Projects

Local Leaders (group project)

Concept: Create an application that could search for all current political leaders in a given address. I was responsible for creating the iframe and connecting wikipedia to it so we could show our search results on our page, the user information input panel, and the design of the page.

$tock Run (group project)

Concept: Create a stock tracking application. I was responsible for creating the chart to display the stocks. This was an extremely frustrating project as my team decided upon using React to create the application and at the time I was brand new to the language. It was also frustrating because the library we decided to use for the chart, Chart.js, worked wonderfully in Javascript/jQuery but presented numerous errors when trying to merge it with React.

Tools: React, CSS, MongoDB, Chart.js


Chart.js

Concept: Learn a new library for using charts in coding. Chart.js is easy to use and learn, and comes equipped with every type of chart the user could need. The two biggest challenges with this project were getting the chart to dynamically update on a set interval, and getting it to apply those updates to every dataset at once regardless of there are 1, 2, 10 or even 0 datasets currently active.

Tools: HTML, CSS, JavaScript, Chart.js
